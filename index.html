Python · OpenCV — March 2025
Hand Detection System
actually this project is just for fun, This project is a Hand Detection System built with Python, leveraging OpenCV and MediaPipe to track and recognize hand landmarks in real time using a webcam. The program detects and maps multiple hand points, allowing the system to recognize gestures such as raised fingers or specific hand positions. It includes a modular structure where the HandDetection class processes frames, extracts landmark data, and evaluates gestures. The system can be extended to trigger actions (like opening applications) based on detected hand gestures, making it a foundation for gesture-based interaction and automation.
Scratch - 1.0 — Mar 16, 2024
ScratchMath 1.0 - Minigame
ScratchMath – 1.0 is an interactive math quiz minigame developed in Scratch that tests players’ calculation skills through multiple difficulty levels. The game allows users to select between easy, medium, and hard modes using keyboard inputs, ranging from basic arithmetic to ranking numbers and randomized challenges. It also incorporates specific answer formatting rules, such as using a dot as a thousands separator (e.g., 100.000), adding an extra layer of precision and attention to detail. Designed with simple controls and a reset feature for replayability, the project demonstrates creativity in game design, logical structuring, and user interaction.
